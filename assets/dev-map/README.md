## Agent 协作关系

| Agent | 上游 | 下游 | 可回退到 |
|-------|------|------|---------|
| PM调度 | 用户 | 需求拆解 | — |
| 需求拆解 | PM调度 | 方案设计 | PM调度 |
| 方案设计 | 需求拆解 | 闸门风控 | 需求拆解 |
| 闸门风控 | 方案设计 | 开发执行 | 方案设计 |
| 开发执行 | 闸门风控 | 代码审查 | 闸门风控/方案设计 |
| 代码审查 | 开发执行 | 测试验收 | 开发执行 |
| 测试验收 | 代码审查 | PM调度(交付) | 开发执行/方案设计/需求拆解 |

## 门禁节点映射

| 流转节点 | 门禁脚本 | 校验内容 |
|---------|---------|---------|
| 需求→设计 | check-spec | SPEC 完整性 |
| 设计→风控 | check-design | 设计合规性 |
| 风控→开发 | check-risk | 风险可控性 |
| 开发→审查 | check-code-style + check-security | 代码质量+安全 |
| 审查→测试 | check-review-pass | 审查通过 |
| 测试→交付 | check-test-coverage | 测试充分性 |
